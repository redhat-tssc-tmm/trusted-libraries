## Verification Process

Each package was verified using the following process:

1. **Query Package Metadata** - Fetch package info from the Red Hat Trusted Libraries index using PEP 691 JSON API
2. **Select Wheel with Attestation** - Find a wheel file that has a provenance URL
3. **Fetch Attestation** - Download the attestation from the integrity API endpoint
4. **Extract Statement & Signature** - Parse the attestation bundle to get the in-toto statement and signature
5. **Create DSSE PAE** - Construct the Pre-Authentication Encoding per DSSE spec:
   ```
   DSSEv1 <type_len> <type> <payload_len> <payload>
   ```
6. **Verify Signature** - Use cosign to verify the RSA signature over the PAE
